## Prerequisites

- **Language-specific tools** - dotnet, rust
- **Text Editor** - [VsCode](https://code.visualstudio.com/) is a great option with language-specific plugins
- **Terminal** - Heim is accessed through its command-line interface

<br />

## Language-specific tools

<Tabs>
    <TabItem label="C-Sharp" icon="seti:c-sharp">
        <LinkButton href="https://dotnet.microsoft.com/en-us/download">Install Dotnet</LinkButton>  <Badge text="requirement: .NET 10.0 preview or higher" variant="caution" />
    </TabItem>
    <TabItem label="Python" icon="seti:python">
        <LinkButton href="https://www.python.org/downloads/">Install Python</LinkButton>
        <Steps>
            <ol>
                <li>
                    Install Python
                    <Tabs syncKey="os">
                        <TabItem label="windows" icon="seti:windows">
                            https://www.python.org/downloads/
                        </TabItem>
                        <TabItem label="macos" icon="apple">
                            <Code lang="sh" frame="none" code="brew install python" />
                        </TabItem>
                        <TabItem label="linux" icon="linux">
                            <Code lang="sh" frame="none" code="sudo apt-get install python3" />
                        </TabItem>
                    </Tabs>
                </li>
                <li>
                    Install componentize-py
                    <Code code="pip install componentize-py"  lang="sh" frame="none" />
                </li>
            </ol>
        </Steps>
    </TabItem>
    <TabItem label="Rust" icon="seti:rust">
        <LinkButton href="https://www.rust-lang.org/tools/install">Install Rust</LinkButton>  <Badge text="requirement: version 1.82 or newer" variant="caution" />
        <Steps>
            <ol>
                <li>
                    Install Rust
                    <Tabs syncKey="os">
                        <TabItem label="windows" icon="seti:windows">
                            https://forge.rust-lang.org/infra/other-installation-methods.html

                            If you don't install msvc via rustup or already have it preinstalled then you need to install msvc.

                            https://rust-lang.github.io/rustup/installation/windows-msvc.html
                        </TabItem>
                        <TabItem label="macos" icon="apple">
                            <Code lang="sh" frame="none" code="curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh" />
                        </TabItem>
                        <TabItem label="linux" icon="linux">
                            <Code lang="sh" frame="none" code="curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh" />
                        </TabItem>
                    </Tabs>
                </li>
                <li>
                    Install wasm target
                    <Code code="rustup target add wasm32-wasip2"  lang="sh" frame="none" />
                </li>
            </ol>
        </Steps>
    </TabItem>
    <TabItem label="Typescript" icon="seti:typescript">
        <LinkButton href="https://nodejs.org/en/download/package-manager">Install NodeJs</LinkButton>
