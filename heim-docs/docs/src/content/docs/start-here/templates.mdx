Templates provide a quick means for creating an application in Heim, without having to do it from scratch.

<br />

## Find out more about templates

If you need information about using templates in Heim, have a look at the [_Getting Started_](./../../start-here/getting-started/#using-applications) document, where you'll find general information about using templates.

<br />

You'll also find more in-depth, practical examples of template use in our basic examples for different programming languages:

 <br />

<Icon name="seti:c-sharp" label="C# language icon" class="inline-icon" /> [Basic C# Application](./../../guides/examples/basic-csharp-application/) guide <br />
<Icon name="seti:python" label="Python language icon" class="inline-icon" /> [Basic Python Application](./../../guides/examples/basic-python-application/) guide <br />
<Icon name="seti:rust" label="Rust language icon" class="inline-icon" /> [Basic Rust Application](./../../guides/examples/basic-rust-application/) guide <br />
<Icon name="seti:typescript" label="TypeScript language icon" class="inline-icon" /> [Basic TypeScript Application](./../../guides/examples/basic-typescript-application/) guide <br />
<Icon name="seti:c" label="C language icon" class="inline-icon" /> [Basic C Application](./../../guides/examples/basic-c-application/) guide <br />
<Icon name="seti:go" label="Go language icon" class="inline-icon" /> [Basic Go Application](./../../guides/examples/basic-go-application/) guide

<br />

<br />

The examples above make use of Heim's _default templates_. These templates come included with every Heim installation.

For a list of built-in default templates that come included with your Heim installation, check out our [_Built-in templates_](./../../templates/built-in/) document.
