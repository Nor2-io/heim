## Resources

### Projects
Maximum number of projects in this organization.


### Applications
Maximum number of applications across all projects.


## Usage

Usage based limits are reset monthly.

### Monthly Requests
Total API requests per month across all applications.

### Outgoing Bandwidth
Total outgoing network bandwidth per month across all applications.


## Team

### Organization Members
Maximum number of members in this organization.


## Application Limits

### HTTP Trigger Execution Time
Maximum execution time per HTTP trigger invocation.

### Cron Trigger Execution Time
Maximum execution time per cron trigger invocation.

### Application Size Limit
Maximum size for application deployments.

### Max Application Memory per Invocation
Maximum memory available per function invocation.
