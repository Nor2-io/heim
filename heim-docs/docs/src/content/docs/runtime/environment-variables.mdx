Quick lookup for every environment variable the Heim runtime reads. For full descriptions, types, and grouping, see [Runtime Configuration](./configuration).

Env variables override `config.toml` and are themselves overridden by CLI arguments. See [Configuration File](./config-file) for the precedence rules.

## CLI environment variables

These are read by the `heim` CLI, not by the runtime process itself.

| Environment variable | Default | Purpose |
| --- | --- | --- |
| `HEIM_CI_TOKEN` | unset | CI deployment token (hidden from `--help`) |
| `HEIM_ORGANIZATION` | unset | Default organization for CLI commands |
| `HEIM_PROJECT` | unset | Default project for CLI commands |
| `GITHUB_CLIENT_ID` | unset | GitHub client ID for self-hosted setups |

`HEIM_RUNTIME_HOST` and `HEIM_RUNTIME_PORT` are also honored by the CLI when targeting a runtime.
